Breadcrumbs (migas de pan)

¿Qué son?

Los breadcrumbs muestran la ruta de navegación del usuario dentro del sitio. Funcionan como una “dirección” (Inicio / Sección / Página) y ayudan a entender dónde estás y cómo volver.

¿Cuándo conviene usarlos?
  • Sitios con varias secciones o navegación jerárquica (categorías, módulos, unidades).
  • Contenido profundo: cuando el usuario entra a páginas internas desde búsqueda o links.
  • UX: para reducir la “sensación de pérdida” al navegar.
Buenas prácticas
  • Inicio siempre primero y con link.
  • Último elemento sin link (es la página actual).
  • Nombres claros: usa etiquetas que coincidan con el menú.
  • Consistencia: mismo separador y estilo en todo el sitio.
  • No reemplaza al menú: es apoyo, no sustituto.
¿Cómo funciona en este proyecto?
  • Se construye a partir de la URL (segmentos) y siempre inicia con Inicio.
  • Se usa un mapa de etiquetas (por ejemplo: mapa-sitio → “Mapa del sitio”, busqueda → “Búsqueda”).
  • Se actualiza al cambiar de página (evento NavigationEnd del router).
Tip: navega por el menú superior y observa cómo cambian las migas de pan en el layout.
